# e-commerce

E-Commerce project. School work

## Build and Install

```bash
npm ci
npm run build
```

## Up

```bash
npm start
```

## Dependencies

- `bcrypt`
- `cookie-parser`
- `cors`
- `dotenv`
- `ejs`
- `express`
- `jsonwebtoken`
- `node-cron`
- `nodemailer`
- `path`
- `sequelize`
- `sqlite3`
- `xss`
- `zod`
